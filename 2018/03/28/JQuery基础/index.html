JQuery基础

发表于 2018-03-28

先前自学了一段时间的JQuery，确实比原始的js简便不少。最近一直在学vue好久没用了，先记录下来，以免忘记。因为只是做个记录，所以内容就不展开了。

（注：带冒号的都是修饰前面的标签）

基础选择器

根据id选择，根据class选择，根据标签选择等等

层级选择器

内容选择器

:has()选择器选择含有选择器所匹配的元素的父元素，返回集合元素
:empty选择器选择不包含子元素或文本的空元素，返回集合元素
:parent选择器选择含有子元素或文本的元素，返回集合元素
:contains(text)选择器选择含有文本内容为’text’的元素，返回集合元素

属性选择器

$(“[attribute|=’value’]”)
*=包含；^=开头是；$=结尾是；

过滤选择器

:first 选取第一个元素,例$(“div:first”)
:last 选取最后一个元素 $(“div:last”)
:not() 去除所有给定选择器匹配的元素,例 $(“input:not(.myClass)”)
:even 选取索引数是偶数的元素 索引从0开始 $(“div:even”)
:odd 选取索引数是奇数的元素 索引从0开始 $(“div:odd”)
:eq(index)选取索引数等于index的元素 $(“div:eq(2)”)
:gt(index)选取索引数大于index的元素 $(“div:gt(2)”)
:lt(index)选取索引数小于index的元素 $(“div:lt(2)”)

子元素选择器

:nth-child(index/evenn/odd/equation)
解析：:eq(index)只匹配一个元素，而:nth-child将为每一个父元素匹配子元素，并且:nth-child(index)的index是从1开始的，而:eq(index)是从0开始的
:first-child 选取每个父元素的第一个子元素 集合元素
:last-child 选取每个父元素的最后一个子元素
:only-child 如果某个元素是他父元素中唯一的子元素，那么将会被匹配。

表单选择器

:input 匹配所有form元素；

表单属性

:checked 选取所有被选中的元素（单选框，复选框
:selected 选取所有被选中的选项元素（下拉列表）
